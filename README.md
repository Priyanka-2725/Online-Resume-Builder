# 🧾 Online Resume Builder

> **Build. Customize. Download.**  
> A powerful, form-based resume generator that helps you craft elegant, professional resumes in minutes — no design skills needed.

---

## 🚀 About the Project

Have you ever struggled to build a resume from scratch or wasted time formatting it to perfection?  
**Online Resume Builder** solves that — offering a seamless, interactive experience where you can simply fill out your details, and generate a polished PDF instantly.

This project is ideal for:
- Students preparing for internships or placements
- Job seekers looking for fast, professional resumes
- Developers learning how to create real-world full-stack projects

---

## ✨ Features

✅ Interactive, form-based interface  
✅ Clean and modern design  
✅ Real-time resume preview  
✅ One-click PDF generation  
✅ Fully responsive and browser-friendly  
✅ Built entirely using **HTML**, **CSS**, **JavaScript**, and **PHP** — no frameworks!

---

## 🧰 Tech Stack

| Tech        | Description                            |
|-------------|----------------------------------------|
| **HTML5**   | Structure of the resume builder         |
| **CSS3**    | Styling with responsiveness             |
| **JavaScript** | Dynamic form behavior & interactions |
| **PHP**     | Backend logic and data processing       |
| **jsPDF/FPDF** | Convert resume to downloadable PDF   |
| **XAMPP**   | Local server for running PHP scripts    |
